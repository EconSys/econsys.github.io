Simulating the Workforce

Workforces are complex, dynamic systems. To project the evolution of current employees, our general approach is to:

  1. Model the potential states for the next period.
  2. Use these model(s) to simulate a new state for each employee.
  3. Update any time-dependent variables and classify each employee into the appropriate model.
  4. Repeat this process for several years into the future over a large number of trials.
